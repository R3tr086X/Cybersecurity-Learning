# Projects / Proyectos

## Español

Esta carpeta contiene pequeños proyectos desarrollados con fines educativos durante clases o de forma autónoma, relacionados con redes, sistemas y seguridad informática.

Aquí encontrarás trabajos que reflejan la aplicación práctica de conceptos aprendidos, incluyendo configuraciones de red, implementación de servicios, prácticas de seguridad y automatización básica.

### Objetivo
- Consolidar conocimientos prácticos en ciberseguridad  
- Documentar proyectos educativos y experimentos técnicos  
- Servir como portfolio técnico organizado y visible  

**Nota:** Todo el contenido es educativo y se ha realizado en entornos controlados (laboratorios y máquinas de práctica).

---

## English

This folder contains small educational projects developed during coursework or independently, focused on networking, systems, and cybersecurity practice.

You will find work that reflects the practical application of learned concepts, including network configurations, service deployment, security exercises, and basic automation.

### Goal
- Build strong practical foundations in cybersecurity  
- Document educational projects and technical experiments  
- Serve as an organized and visible technical portfolio  

**Disclaimer:** All content is for educational purposes only and was created in controlled lab environments.
